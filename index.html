6-Day Workout Plan (Dumbbells ke Saath)

Day 1: Chest & Back

  • 1. Diamond Push-Ups: 3 sets x 10-15 reps
  • 2. Incline Push-Ups: 3 sets x 10-15 reps
  • 3. Decline Push-Ups: 3 sets x 10-15 reps
  • 4. Wide Push-Ups: 3 sets x 10-15 reps
  • 5. Dumbbell Chest Press: 3 sets x 10-15 reps
  • 6. Dumbbell Bent Over Row: 3 sets x 10-15 reps
  • 7. Pull-Ups: 3 sets x 5-10 reps

Day 2: Legs & Shoulders

  • 1. Squats: 3 sets x 10-15 reps
  • 2. Dumbbell Lunges: 3 sets x 10-15 reps (each leg)
  • 3. Dumbbell Shoulder Press: 3 sets x 10-15 reps
  • 4. Lateral Raise: 3 sets x 10-15 reps

Day 3: Arms & Forearms

  • 1. Dumbbell Bicep Curl: 3 sets x 10-15 reps
  • 2. Tricep Kickback: 3 sets x 10-15 reps
  • 3. Hammer Curl: 3 sets x 10-15 reps
  • 4. Dumbbell Wrist Curls: 3 sets x 15-20 reps
  • 5. Reverse Dumbbell Wrist Curls: 3 sets x 15-20 reps

Day 4: Chest & Back (Light)

  • 1. Incline Push-Ups: 3 sets x 10-15 reps
  • 2. Dumbbell Chest Fly: 3 sets x 10-15 reps
  • 3. Dumbbell Deadlift: 3 sets x 10-15 reps
  • 4. Dumbbell Shrug: 3 sets x 10-15 reps

Day 5: Legs, Calves & Shoulders (Light)

  • 1. Goblet Squat: 3 sets x 10-15 reps
  • 2. Standing Calf Raises: 3 sets x 15-20 reps
  • 3. Dumbbell Front Raise: 3 sets x 10-15 reps
  • 4. Rear Lateral Raise: 3 sets x 10-15 reps

Day 6: Core & Obliques

  • 1. Plank: 3 sets x 30-60 seconds
  • 2. Russian Twists: 3 sets x 15-20 reps (each side)
  • 3. Side Plank: 3 sets x 30 seconds (each side)
  • 4. Bicycle Crunches: 3 sets x 15-20 reps

Day 7: Rest Day

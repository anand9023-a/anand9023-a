Eye Exercises

1. Pencil Push-ups

  • Repetitions:
  • 10-15 reps per set, 2 sets
  • Duration:
  • Hold each repetition for 5-10 seconds

2. Brock String Exercise

  • Repetitions:
  • 10-12 reps, 2 sets
  • Duration:
  • Hold focus on each object for 10-15 seconds

3. Focus Shifting

  • Repetitions:
  • 10-15 reps per set, 2 sets
  • Duration:
  • Hold focus for 5-10 seconds per repetition

4. Near and Far Focusing

  • Repetitions:
  • 10-12 reps, 2 sets
  • Duration:
  • Hold focus for 5-10 seconds per repetition

5. Eye Rotation

  • Repetitions:
  • 5-10 rotations clockwise and counter-clockwise
  • Duration:
  • 10-15 seconds per rotation

6. Palming

  • Repetitions:
  • 2-3 sets
  • Duration:
  • 1-2 minutes per set

Note: Do these exercises daily for 10-15 minutes to see improvement in your eye health and appearance.
